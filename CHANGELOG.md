# Changelog

All notable changes to the Diné Insurance website will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-07-17

### Added
- Initial release of Diné Insurance website
- Homepage with hero section featuring Navajo Code Talker heritage messaging
- Core values section highlighting Hózhó, K'é, and traditional protection principles
- Services overview with homeowners, auto, business, and life insurance
- About page with mission, values, and team information
- Services page with detailed coverage information
- Contact page with quote request form and office locations
- Blog system with MDX support for educational content
- Sample blog posts about insurance topics relevant to Diné families
- Responsive design optimized for mobile and desktop
- Cultural color scheme (Red Clay, Turquoise, Sand)
- Typography system with Playfair Display and Inter fonts
- Custom Tailwind CSS configuration with Diné-inspired design tokens
- SEO optimization with proper metadata and sitemap generation
- Netlify deployment configuration
- Accessibility features and semantic HTML structure

### Technical Features
- Next.js 15 with App Router architecture
- TypeScript for type safety
- Tailwind CSS with custom design system
- Radix UI components for accessibility
- MDX blog system with gray-matter frontmatter parsing
- Lucide React icons with mountain and cultural iconography
- Reading time calculation for blog posts
- Automatic sitemap generation including blog posts
- Performance optimizations with Next.js image optimization

### Cultural Elements
- Respectful integration of Diné language terms (Hózhó, K'é)
- Four Sacred Mountains symbolism in design and messaging
- Navajo Code Talker heritage honored in hero messaging
- Traditional building and cultural practices referenced in insurance context
- Community-focused testimonials and case studies
- Culturally sensitive color palette and imagery

### Content
- "Understanding Homeowners Insurance on Navajo Nation" blog post
- "Protecting Your Traditional Crafts: Business Insurance for Diné Artisans" blog post
- "Navigating Auto Insurance on Navajo Nation: Coverage for Reservation Roads" blog post
- Comprehensive service descriptions for all insurance types
- Contact information and office locations
- Team member profiles with clan affiliations
- Mission statement and company values

### Infrastructure
- GitHub repository setup with proper branch structure
- Netlify deployment with custom domain configuration
- Build optimization for static site generation
- Error handling and loading states
- Mobile-responsive navigation with hamburger menu
- Footer with comprehensive site navigation
